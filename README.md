 🧠 Neural Network

A collection of Deep Learning projects implemented using **TensorFlow/Keras** in Google Colab. This repository demonstrates different neural network architectures for image classification, from a simple Deep Neural Network (DNN) to advanced Transfer Learning models such as ResNet50, EfficientNet-B0, and Vision Transformer (ViT).

---

 📂 Repository Structure

```
Neural_Network/

├── dnn.ipynb
├── dnn(regularization,_dropout,_batchnormalization).ipynb

├── cnn(EfficientNet-B0).ipynb

```

How to Run

1. Clone the repository

```bash
git clone https://github.com/rajasekar-123/Neural_Network.git
```

2. Open any notebook in Google Colab or Jupyter Notebook.

3. Install dependencies

```bash
pip install tensorflow numpy matplotlib
```

4. Run all cells.
